📖 Guia Rápido

Como usar o Editor:

  1. Escolha um template base ou crie do zero
  2. Edite o HTML no editor
  3. Use placeholders como {{nome_campo}}
  4. Visualize o preview com dados de exemplo
  5. Baixe o template ou exporte como PDF

Placeholders Comuns:

  • {{data}} - Data
  • {{hora}} - Hora
  • {{empresa}} - Nome da empresa
  • {{participantes}} - Lista de participantes
  • {{decisoes}} - Decisões tomadas

Dicas:

  • Use HTML semântico (<section>, <article>)
  • Mantenha CSS inline ou na tag <style>
  • Teste o preview frequentemente
  • Campos com listas devem usar HTML: <li>item</li>
